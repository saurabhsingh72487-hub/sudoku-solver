# 🧩 Sudoku Solver

A full-stack Sudoku Solver application built with Node.js, Express, and JavaScript.  
This project solves Sudoku puzzles and validates placements using custom solving logic and API endpoints.

---
## 🌐 Live Demo

👉 https://sudoku-solver-yvmm.onrender.com/
## 🚀 Features

- Solve Sudoku puzzles
- Validate puzzle placements
- REST API support
- Full backend validation
- Unit & Functional tests
- Responsive frontend UI

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- JavaScript
- Mocha
- Chai
